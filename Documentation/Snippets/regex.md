## Examples

### Find five digit numbers



| Syntax | Description |
| ----------- | ----------- |
| `^\d{5}$` | Find lines that are a 5 Digit number |
| `^(.*)\d{5}(.*)$` | Finds 5 Digit Numbers inside other characters in a line |
